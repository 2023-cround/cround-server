# 크라운드 - 백엔드

### 프로젝트 주요 관심사

- 도메인 모델 패턴에 맞춘 설계와 값 객체, 일급 컬렉션을 활용한 객체 지향적인 설계
- 계층형 아키텍처의 역할을 명확히 분리하여 패키지로 분리하고, 의존성을 최소화하여 유연한 아키텍처로 구축
